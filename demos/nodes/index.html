Nodes SDL2/emscripten Demo

Libraries used: SDL2, SDL2_gfx, stb_image, emscripten.

Controls:

  • Click on a node to select it. Click and drag to move it around
  • Once a node is selected, click anywhere to insert a node connected to the selected node, if an existing node is clicked, a new directed segment is inserted.
  • Press delete on a selected node to delete it
